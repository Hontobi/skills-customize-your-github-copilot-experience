# 📘 Assignment: Python Text Processing

## 🎯 Objective

Master working with strings, file I/O operations, and text manipulation in Python. You'll learn to read and process text files, manipulate strings effectively, and transform data for practical applications.

## 📝 Tasks

### 🛠️ Process and Analyze Text Files

#### Description
Build a text processing program that reads a text file, manipulates the content, and outputs processed results. Your program should handle string operations, file reading/writing, and text transformation techniques.

#### Requirements
Completed program should:

- Read text from a file and handle file I/O operations correctly
- Perform string manipulation (splitting, joining, searching, replacing)
- Count and analyze text statistics (word count, character count, line count)
- Transform and format text output
- Write processed results to an output file


### 🛠️ String Analysis and Filtering

#### Description
Create a program that analyzes text content to find, filter, and manipulate specific strings based on user-defined criteria. This task focuses on pattern matching and string searching techniques.

#### Requirements
Completed program should:

- Search for specific words or patterns in text
- Filter lines or words based on conditions
- Convert text between different formats (uppercase, lowercase, title case)
- Remove unwanted characters or whitespace
- Display summary statistics about the processed text
